# CVP (Car Value Planner)

CVP is a static add-on tool served by CorePortal at `/CVP`.

Key features:
- Out-the-door purchase breakdown
- Financing estimate (down payment, APR, term)
- Tier-aware planning (economy, practical, comfort, sport, luxury, exotic)
- Monthly affordability guidance

Files:
- `cvp_planner.html` - main planner UI
- `policy_years.json` - tax/fee policy presets

Compatibility:
- Legacy `/OTD` routes are redirected to `/CVP`
